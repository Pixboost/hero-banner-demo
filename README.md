# Hero Banner Demo

Hero banner demo for the blog article that show how well implemented hero banner
could improve LCP and CLS Core Web Vital's metrics

## Running the demo

```shell
docker run -p 8080:80 -v $(pwd):/usr/share/caddy caddy:2.3.0-alpine
```
